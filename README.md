## A Mean Stack app that allows you schedule a styling appointment with a BR stylist in your area.

(Brief descrption of the problem i am solving for BR.)

<img src="" width="100%">


## User Story

1. A user can find a stylist by entering their zip code or hsaring their location.
2. A user can search stylists by name and location and schedule an appointment.
3. An admin can create, delete, and update, a stylist from the admin portal.
